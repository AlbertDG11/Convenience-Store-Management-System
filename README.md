Convenience Store Database Management System


Initialise the project
Backend (Django):
`python -m venv venv`
`.venv\Scripts\activate`
`pip install -r requirements.txt`
`python manage.py migrate`

Frontend:
`cd frontend`
`npm install`


Running the project:
Backend (Django):
`python manage.py runserver`

Frontend (React):
`cd frontend`
`npm start`
